# Интернет магазин  
**Направление подготовки:** 09.03.04 - Программная инженерия   
**Сроки прохождения практики:** 04.02.26 - 26.05.26  
**Руководитель практики:** ст. преподаватель В. М. Димитров 

Проект интеренет магазина на Fastapi и react с применением sqlalchemy, redux, and Postgresql

## Реализованный функционал

### Товары

* [x] Добавление товара (`POST /api/product/`)
* [x] Получение всех товаров (`GET /api/product/`)
* [x] Получение одного товара (`GET /api/product/{productid}`)
* [x] Обновление товара (`PUT /api/product/{productid}`)
* [x] Удаление товара (`DELETE /api/product/{productid}`)
* [ ] Пагинация и поиск товаров (в разработке)

### Пользователи и роли

* [x] Регистрация и получение списка пользователей (`/api/users/`)
* [x] Авторизация через JWT (`/api/login`)
* [ ] Полная поддержка ролей в интерфейсе (администратор, менеджер, обычный пользователь)

### Заказы

* [x] Создание заказа без оплаты (`POST /api/order/`)
* [x] Просмотр всех заказов и заказов пользователя (`GET /api/order/`)
* [x] Удаление заказа (`DELETE /api/order/orderbyid/{id}`)
* [x] Отправка e-mail уведомлений менеджерам о новом заказе
* [ ] Реальные уведомления через WebSocket

### Отзывы

* [x] Создание и получение отзывов о товарах (`/api/review/`)

### Прочее

* [x] ORM на SQLAlchemy
* [x] OpenAPI документация доступна на `/openapi.json`
* [x] Подготовка к интеграции с Stripe

---

## Планируемый функционал (в разработке)

* Категории товаров (CRUD + связь с товарами)
* Бренды товаров (CRUD + связь с товарами)
* Изображения товаров (CRUD)
* Пагинация и поиск товаров
* Интерфейс админки (React/Vue + MUI/Vuetify)
* Уведомления менеджера о заказе в реальном времени через WebSocket
* Управление ролями пользователей через UI

## Запуск
python3 -m uvicorn main:app --reload --port 8000

Магазин будет доступен по адресу: `http://localhost:8000/docs`


## Структура проекта

app/
├── auth/               # Аутентификация и JWT
├── config/             # Настройки БД и приложения
├── models/             # SQLAlchemy модели
├── order/              # Работа с заказами
├── product/            # Работа с товарами
├── review/             # Работа с отзывами
├── schemas/            # Pydantic схемы
├── users/              # Работа с пользователями
└── main.py             # Точка входа FastAPI
